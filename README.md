# Planner

A customisable weekly planner webapp built with React, Vite and TypeScript.

Configure your schedule, add recurrent activities, then print your planner.

## Stack

- **React 19** + **Vite** + **TypeScript**
- **Material UI (MUI v7)** – accessible component library
- **react-hook-form v7** – form state and validation
- **React Router v7** – client-side routing
- **Vitest** + **Testing Library** – unit and component tests

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173/planner/](http://localhost:5173/planner/) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run lint` | ESLint |
| `npm test` | Run tests |
| `npm run preview` | Preview production build |
