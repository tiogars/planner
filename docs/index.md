# Planner

A customisable weekly planner webapp. Configure your time slots and days, define recurrent activities, then print your planner.

## Features

- **Setup** – choose the planner title, active days and time range
- **Activities** – add recurring activities with colour, time slot and days
- **Print** – preview and print the populated weekly grid

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173/planner/](http://localhost:5173/planner/) in your browser.
